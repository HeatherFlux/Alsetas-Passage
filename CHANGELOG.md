# Changelog

All notable changes to Alseta's Passage will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-01-27

### Added
- ES modules support for better code organization
- Modular architecture with separate utility and feature modules
- Improved build system with versioned packages
- Enhanced error handling and logging
- Toast notifications for user feedback
- Support for multiple Discord webhooks
- Better character data extraction

### Changed
- Refactored codebase for better maintainability
- Improved message formatting for Discord
- Enhanced dynamic content handling
- Better separation of concerns in code structure

### Fixed
- Event listener duplication issues
- Character name tracking improvements
- HTML sanitization for better security

## [1.0.0] - Initial Release

### Added
- Basic integration between Pathbuilder2e and Discord
- Support for dice rolls and character actions
- Discord webhook configuration
- "Send to Discord" buttons on Pathbuilder interface
- Support for both main and beta Pathbuilder versions
